# 图片处理工具

Windows 桌面图片处理软件。用户直接导入一张或多张图片，通过左侧导航选择处理功能。

作者：Konsheng

版本：v1.0.0

## 功能

- 左侧可折叠导航栏
- 综合处理
- 尺寸处理
- 格式转换
- 图片压缩
- LOGO叠加
- LOGO 素材库多选叠加
- 水印
- 处理后效果预览
- 批量处理取消
- 浅色、暗色、跟随系统主题
- 关于页
- 支持拖拽导入
- 支持 JPG、PNG、WEBP、保持原格式
- 支持 1MB、2MB、3MB、自定义压缩大小
- 输出文件不覆盖原图，重名时自动加序号

## 输出尺寸

综合处理和尺寸处理支持以下预设规格：

- 800×800 px
- 1000×1000 px
- 1200×1200 px
- 1440×1440 px
- 1920×1920 px
- 自定义宽高

默认规格为 1440×1440 px。尺寸处理会等比缩放图片，不裁切主体，不足区域使用白色背景补齐。

## 独立功能页

- 综合处理：尺寸、格式、压缩、LOGO、水印可组合设置
- 尺寸处理：只调整尺寸，保持原格式；BMP 输出为 JPG
- 格式转换：只转换格式，保持原尺寸
- 图片压缩：只压缩，保持原尺寸和原格式；BMP 输出为 JPG
- LOGO叠加：从内置素材库多选叠加图，保持原尺寸和原格式；BMP 输出为 JPG
- 水印：添加文字水印或图片水印，保持原尺寸和原格式；BMP 输出为 JPG

## LOGO 素材库

内置 LOGO/叠加图素材文件夹：

```text
assets/logos/
```

支持格式：

- PNG
- JPG
- JPEG
- WEBP

透明 PNG 效果最好。素材按文件名排序后显示为大图网格，用户可以多选；处理时会按网格顺序逐层叠加。素材会自动缩放到当前图片尺寸，并从 `(0, 0)` 直接覆盖合成。

旧路径 `assets/logo.png` 会作为兼容备用资源保留。新版优先读取 `assets/logos/`。

## 水印

`水印` 页面只执行水印叠加。`综合处理` 页面可以勾选 `添加水印`，处理顺序为：尺寸处理、LOGO叠加、水印、格式转换、压缩保存。

水印类型：

- 文字水印：支持文字内容、字号、颜色、透明度、九宫格位置、自定义坐标、边距、旋转角度、平铺
- 图片水印：选择本地水印图片，支持缩放比例、透明度、九宫格位置、自定义坐标、边距、旋转角度、平铺

默认文字为 `Konsheng`，默认位置为右下，默认透明度为 35%，默认边距为 40 px，默认角度为 0 度。输入 `-30` 或 `30` 可以生成斜向水印。

水印可以在右侧预览中拖拽定位。非平铺水印拖拽后会切换为自定义坐标；平铺水印拖拽后会调整平铺起始位置。平铺模式支持水平间距、垂直间距、起始X、起始Y。

## 预览与取消

每个功能页右侧显示全高固定预览侧栏。左侧设置和处理结果可以滚动，右侧预览不会跟随滚动离开视口。预览侧栏支持动画折叠，折叠后右侧保留窄栏展开按钮，折叠状态在所有功能页之间同步。导入并选中图片后，预览会按当前页面设置显示处理后效果。预览只在内存中生成，不保存文件，也不模拟压缩后的质量变化。

批量处理时可以点击 `取消处理`。软件会等待当前图片处理完成，再停止后续任务；未开始的图片会标记为 `已取消`。

## 主题

左侧导航底部提供 `设置` 页面。主题支持：

- 浅色
- 暗色
- 跟随系统

主题选择会保存到当前 Windows 用户配置中，下次打开软件时自动恢复。

## 关于

左侧导航底部提供 `关于` 页面，显示软件名、作者、版本和核心功能。

## 运行

```bat
pip install -r requirements.txt
python main.py
```

## 打包

确认 `assets/logos/` 中至少有一个支持格式的素材后，执行：

```bat
build.bat
```

打包命令使用：

```bat
python -m PyInstaller --noconfirm --clean image_tool.spec
```

输出文件位于：

```text
dist/图片处理工具.exe
```

## GitHub Actions 发布

仓库包含 Release 工作流，支持推送 `v*` 标签自动发布，也支持在 GitHub Actions 页面手动运行。

自动发布示例：

```bat
git tag v1.0.0
git push origin v1.0.0
```

手动发布：

1. 打开 GitHub 仓库的 Actions 页面
2. 选择 `Release`
3. 点击 `Run workflow`
4. 输入版本标签，例如 `v1.0.1`

Release 会上传：

- `图片处理工具.exe`
- `图片处理工具-版本号-windows.zip`
